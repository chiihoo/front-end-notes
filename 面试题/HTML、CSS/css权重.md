|                             样式                             | 权重 |
| :----------------------------------------------------------: | ---- |
|                         `!important`                         | 最高 |
|                           内联样式                           | 1000 |
|                           id选择器                           | 100  |
| 类选择器`.class`、伪类选择器`:hover`,`:nth-child`、属性选择器`[type等]` | 10   |
| 元素选择器`span`、伪元素选择器`::before`,`::after`,`::first-letter`,`::first-line` | 1    |

`:not()`选择器是不参与优先级计算的，但是这个括号里面的选择器是参与优先级计算的

而像通用选择器`*`，子选择器`>`，相邻选择器`+`，权值为0
