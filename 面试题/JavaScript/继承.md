```
1. 原型链继承
父类的引用属性会在子类实例间共享；子类实例化时不能给父类传参
A.prototype = new B()

2. 构造函数继承
不能继承父类原型上的方法
function A(name) {
  B.call(this, name)
}

3. 组合继承
每个子类实例都有父类的方法，并且也继承了父类的原型
但是相当于实例和原型上都有一份父类的属性方法
function A(name) {
  B.call(this, name)
}
A.prototype = new B()

4. 寄生组合继承
通过Object.create(B.prototype)新建一个父类的原型，并将该原型的constructor指向子类
function A(name) {
  B.call(this, name)
}
A.prototype = Object.create(B.prototype)
A.prototype.constructor = A
```
